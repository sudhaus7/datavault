Installation
============

.. important::

   Use the versions 8.x for TYPO3 CMS 8.7.x

The extension needs to be installed as any other extension of TYPO3 CMS:

#. Switch to the module “Extension Manager”.

#. Get the extension

   #. **Get it from the Extension Manager:** Press the “Retrieve/Update”
      button and search for the extension key *guard7* and import the
      extension from the repository.

   #. **Get it from typo3.org:** You can always get current version from
      `http://typo3.org/extensions/repository/view/guard7/current/
      <http://typo3.org/extensions/repository/view/guard7/current/>`_ by
      downloading either the t3x or zip version. Upload
      the file afterwards in the Extension Manager.

   #. **Use composer**: Use `composer require sudhaus7/guard7`.

#. The Extension Manager offers some basic configuration which is
   explained :ref:`here <extensionManager>`.

Latest version from git
-----------------------
You can get the latest version from git by using the git command:

.. code-block:: bash

   git clone git@github.com:sudhaus7/guard7.git
